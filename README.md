# Hotel Rishi Gardens

A modern, fully responsive, and dynamic hotel management website designed for a seamless user experience. This project leverages vanilla web technologies to deliver high performance, smooth animations, and a functional booking/blogging system without the overhead of heavy frameworks.

**Live Demo:** [https://hotelrishigardens.netlify.app/](https://hotelrishigardens.netlify.app/)

## 🚀 Features

* **Fully Responsive Design:** Optimized for mobile, tablet, and desktop devices to ensure a consistent experience across all screen sizes.
* **Dynamic Content Management:** Built with Vanilla JavaScript to allow the owner to easily update blog posts and booking details, keeping the site current and functional.
* **High-End Visuals:** Includes over 15 custom animations and transitions for a premium, high-quality "Smooth UX" feel.
* **Ready-to-Use Functional UI:** No dead links or placeholder pages; the site is fully operational from navigation to external integrations.
* **Smart Contact System:** Features a custom contact form that auto-prepares a professional email draft for the user, simplifying the communication process between clients and the owner.
* **Social Sharing Integration:** Includes a built-in share functionality, allowing users to instantly send blog details via their preferred messaging platforms.
* **Intuitive Navigation:** A clean, 4-page structure designed for ease of use and quick access to information.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Animations:** Custom CSS Keyframes / JS-driven transitions
* **Deployment:** Netlify and Vercel
* **Dynamic Data:** JS-based local/remote data handling for blogs and bookings

## 📂 Project Structure

The website consists of four primary pages:

1. **Home:** Overview of the hotel, featured services, and high-impact visual banners.
2. **Blog:** Dynamic section for news, updates, and events related to the hotel.
3. **Booking:** Functional interface for guests to check availability and details.
4. **Contact:** Interactive form with automated mail-to functionality.
